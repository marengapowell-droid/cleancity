## 📌 Overview

CleanCity is a **complete waste management system** designed to improve urban sanitation by connecting residents, waste collectors, and municipal authorities on a single platform. Built as an academic project for **Object Oriented Programming (ECCI/ECSI 3205)** at the Technical University of Kenya.

## 🎯 Problem Statement

In cities like Nairobi, waste collection is often inconsistent, leading to:
- ❌ Missed or delayed waste pickups
- ❌ Poor communication between residents and collectors
- ❌ No accountability or tracking
- ❌ Limited recycling awareness
- ❌ Manual, paper-based systems

## ✅ Solution

CleanCity digitizes the entire waste management ecosystem with:

| Feature | Description |
|---------|-------------|
| **👤 Resident Portal** | Request pickups, report issues, track status |
| **🚛 Collector Portal** | View assigned pickups, confirm collections, update status |
| **👑 Admin Dashboard** | Monitor all users, resolve reports, manage collectors |
| **📍 Zone-Based Assignment** | Pickups auto-assigned to collectors by zone (CBD, Westlands, Kilimani, Eastlands, Karen) |
| **📊 Real-time Statistics** | Live updates on pickups, pending reports, resolution rates |
| **🌙 Dark Mode** | Toggle between light and dark themes |
